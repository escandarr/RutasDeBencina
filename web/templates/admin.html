Gestión de Datos

Mapa Acerca de

Dispara las tareas de extracción e importación para mantener los datos de estaciones, precios y promociones al día. Cada acción se ejecuta secuencialmente y muestra su resultado al finalizar.

Los logs se recortan automáticamente para evitar salidas excesivamente largas. Revisa el servidor para obtener trazas completas si es necesario.

Actualizar CNE

Ejecuta la extracción de estaciones y precios desde la API de la CNE, luego importa los datos al esquema metadata.

Actualizar Promociones

Ejecuta las extracciones de promociones (aliados + estáticos) y carga las promociones resultantes en la base de datos.

Historial reciente

Los resultados de la ejecución más reciente se muestran aquí. La vista no guarda estado; los datos se borrarán al recargar la página.
